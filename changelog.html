SpotiSaber 1.3 Update

Version 1.3

Fixed and added the following features!

  • Fixed the broken song selector implemented in 1.2
  • Fixed the 100 song limit! (Actually)
  • Added a version checker in the settings
  • Small UI Changes

Quest Users, I don't own a quest, apparently you need the songs Un-Zipepd. There is a folder called SpotiSaber Zip Cache that will be created in the selected directory. Those zip files will NOT be deleted automatically, an unzip switch is now available in the settings window to avoid extra un-needed files.
